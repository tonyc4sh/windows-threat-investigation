# Windows Threat Investigation using Procmon

This repository contains my independent forensic analysis of a Windows security incident based on a Procmon capture provided as part of a cybersecurity technical assessment.

## Objectives

- Investigate suspicious Windows activity
- Reconstruct attacker timeline
- Identify Indicators of Compromise
- Document attacker TTPs
- Produce an executive incident report

## Investigation Scope

- Process tree analysis
- Command-line analysis
- Timeline reconstruction
- IOC extraction
- Living-off-the-Land detection
- Windows artifact analysis

## Tools

- Microsoft Procmon
- Windows Sysinternals

## Deliverables

- Executive report
- IOC list
- Investigation findings
- Recommendations

## Note

The dataset analysed in this project originated from a publicly distributed technical assessment.

The purpose of this repository is to demonstrate my investigation methodology, documentation skills and incident analysis process.
